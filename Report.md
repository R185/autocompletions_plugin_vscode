# How to add own extension in VSCode 

VSCode is frindly to users, who want to add new extension fo personal purposes.

If u want to be one of them u should know how to write in TypeScript or Javascript. 

Official FAQ use as advice trying using yo code extension generator. Process of using is very simply:

1) You have to choose properties of extensions like name, id, language etc.

2) After that yocode will create project with this properties.

3) Then you have to write ur extension in TypeScript or Javascript in src files.

4) All properties like external commands, startp events, etc. can be changed in package.json.

5) You can check how does ur extension work using node.js debugger in VScode, that will create a window with ur extension working on it.

6) Also u have many ways to realise something using VSCode API provided by Microsoft

That's all!
